# 基础版首轮目录结构设计

## 1. 设计概述

本文档定义基础版商品展示网站首轮实现的目录结构，基于当前已确认技术栈：
- 前端：Next.js + React + TypeScript + Tailwind CSS
- 后端：Spring Boot + JWT
- 数据库：PostgreSQL

## 2. 根目录结构

```text
sell_web/
├── frontend/                   # Next.js 前端工程
├── backend/                    # Spring Boot 后端工程
├── database/                   # PostgreSQL 迁移与初始化脚本

├── .claude/                    # Claude 任务与进度

```

## 6. 目录职责说明

### 6.1 frontend
- 负责前台展示和后台管理界面
- 通过 API 调用后端 Spring Boot
- 不直接处理数据库

### 6.2 backend
- 负责认证、内容管理 API、上传、业务逻辑
- 通过 JPA / JDBC 访问 PostgreSQL
- 提供后台所需 REST 接口

### 6.3 database
- 负责 PostgreSQL 初始化与迁移
- 与后端 `resources/db/migration` 保持一致的迁移策略
