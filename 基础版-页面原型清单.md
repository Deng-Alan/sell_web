# 基础版页面原型清单

## 1. 前台页面
### 1.1 首页
#### 页面目标
- 展示网站定位
- 快速展示核心商品
- 引导访客进入分类或联系商家

#### 页面模块
- 顶部导航
- Banner 区
- 网站简介区
- 公告区
- 推荐商品区
- 分类入口区
- 联系方式区
- 页脚

#### 关键元素
- Logo
- 网站名称
- Banner 图片
- 简介文案
- 公告内容
- 推荐商品卡片
- 咨询按钮

### 1.2 商品列表页
#### 页面目标
- 让访客快速筛选和浏览商品

#### 页面模块
- 顶部导航
- 分类筛选区
- 搜索区
- 商品列表区
- 分页区
- 页脚

#### 商品卡片元素
- 商品封面图
- 商品标题
- 简短说明
- 价格
- 库存
- 咨询/购买按钮

### 1.3 商品详情页
#### 页面目标
- 完整展示单个商品信息
- 引导访客立即联系商家

#### 页面模块
- 面包屑导航
- 商品主图/轮播图
- 商品基础信息区
- 商品详细说明区
- 联系方式区
- 底部推荐商品区，可选

#### 关键元素
- 商品标题
- 商品图片
- 价格
- 库存
- 商品简介
- 图文详情
- 咨询按钮
- 购买按钮
- 联系方式或二维码

### 1.4 联系方式页，可选
#### 页面目标
- 统一展示联系商家的所有方式

#### 页面模块
- 联系方式列表
- 二维码展示区
- 跳转按钮

## 2. 后台页面
### 2.1 后台登录页
#### 页面目标
- 让管理员安全进入后台

#### 页面模块
- 登录表单
- 账号输入框
- 密码输入框
- 登录按钮

### 2.2 后台首页
#### 页面目标
- 作为后台入口页
- 展示快捷操作

#### 页面模块
- 侧边导航
- 顶部栏
- 快捷入口卡片

#### 快捷入口建议
- 商品管理
- 分类管理
- 首页配置
- 联系方式配置

### 2.3 商品管理页
#### 页面目标
- 管理商品的增删改查和状态

#### 页面模块
- 商品搜索区
- 商品表格列表
- 新增按钮
- 编辑按钮
- 删除按钮
- 上下架开关

#### 列表字段建议
- ID
- 商品名称
- 分类
- 价格
- 库存
- 推荐状态
- 上架状态
- 排序
- 更新时间

### 2.4 商品编辑页
#### 页面目标
- 新增或编辑商品内容

#### 表单字段
- 商品名称
- 商品分类
- 商品封面图
- 商品轮播图，可选
- 简短说明
- 商品详情
- 价格
- 库存
- 默认联系方式
- 是否推荐
- 排序值
- 上架状态

### 2.5 分类管理页
#### 页面目标
- 管理商品分类

#### 页面模块
- 分类列表
- 新增分类
- 编辑分类
- 删除分类
- 排序设置

### 2.6 首页内容配置页
#### 页面目标
- 管理首页显示内容

#### 配置项
- Banner 图
- 首页简介
- 公告内容
- 推荐商品
- SEO 标题
- SEO 关键词
- SEO 描述

### 2.7 联系方式配置页
#### 页面目标
- 管理前台的联系入口

#### 配置项
- 联系方式类型
- 显示名称
- 联系值
- 跳转链接
- 二维码图片
- 是否启用
- 展示顺序

## 3. 原型优先级建议
### 第一优先级
- 首页
- 商品列表页
- 商品详情页
- 后台登录页
- 商品管理页
- 商品编辑页

### 第二优先级
- 分类管理页
- 首页内容配置页
- 联系方式配置页

### 第三优先级
- 联系方式独立页
- 后台首页优化版

## 4. 输出建议
- 客户确认阶段：低保真线框图即可
- 开发阶段：补齐字段、按钮状态、弹窗和表单校验规则
- UI 阶段：再统一风格、配色和组件细节
