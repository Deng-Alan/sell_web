# Change: Refine public UI readability

## Why
The current public UI still behaves too much like a promotional poster layout. Product details place readable text on top of strong images, purchase information is visually secondary, and the contact page still feels like a styled showcase instead of a simple manual-buy flow.

## What Changes
- Redesign the public product detail page into a readable image-and-content split layout
- Simplify product cards so they scan like standard commerce cards instead of poster thumbnails
- Simplify the public contact page into stable QR-based purchase cards for primary channels
- Remove leftover English UI labels from the public purchase flow

## Impact
- Affected specs: product-showcase, contact-routing
- Affected code: homepage catalog cards, product detail page, contact page
