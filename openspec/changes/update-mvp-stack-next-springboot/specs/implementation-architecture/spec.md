## ADDED Requirements
### Requirement: MVP implementation architecture baseline
The system SHALL use a Next.js frontend, a Spring Boot backend, PostgreSQL as the primary database, and JWT for admin authentication.

#### Scenario: Planning baseline is updated
- **WHEN** the implementation architecture is documented for the MVP
- **THEN** the documented baseline uses Next.js for frontend pages, Spring Boot for backend APIs, PostgreSQL for relational data, and JWT for admin authentication

#### Scenario: Bootstrap work begins
- **WHEN** the project initializes executable frontend and backend skeletons
- **THEN** the generated skeleton matches the documented Next.js and Spring Boot architecture instead of the previous Nuxt and Express baseline
