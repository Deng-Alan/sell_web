# Change: Update MVP stack to Next.js and Spring Boot

## Why
The implementation baseline previously assumed Nuxt, Express, and MySQL, but the required stack has now changed. The project must align with the new frontend, backend, authentication, and database requirements before further implementation.

## What Changes
- Replace the frontend baseline with Next.js
- Replace the backend baseline with Spring Boot
- Replace the database baseline with PostgreSQL
- Keep JWT as the admin authentication mechanism
- Re-open implementation bootstrap work so the repository is initialized against the new stack

## Impact
- Affected specs: implementation-architecture
- Affected code: frontend bootstrap, backend bootstrap, database migrations, planning docs, task ledger
