# 基础版数据库设计初稿

## 1. 设计说明
本期为基础展示版，数据库以内容管理和商品展示为主，不包含支付、订单、自动发货。

## 2. 数据表清单
- `admin_users` 管理员表
- `product_categories` 商品分类表
- `products` 商品表
- `product_images` 商品图片表
- `contacts` 联系方式表
- `site_settings` 站点配置表
- `home_sections` 首页模块配置表

## 3. 表结构建议
### 3.1 admin_users
| 字段名 | 类型 | 说明 |
|---|---|---|
| id | bigint | 主键 |
| username | varchar(50) | 管理员账号 |
| password_hash | varchar(255) | 密码哈希 |
| nickname | varchar(50) | 昵称 |
| status | tinyint | 状态，1启用0禁用 |
| last_login_at | datetime | 最后登录时间 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

### 3.2 product_categories
| 字段名 | 类型 | 说明 |
|---|---|---|
| id | bigint | 主键 |
| name | varchar(100) | 分类名称 |
| slug | varchar(100) | 分类标识，可选 |
| sort_order | int | 排序值 |
| status | tinyint | 状态，1启用0禁用 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

### 3.3 products
| 字段名 | 类型 | 说明 |
|---|---|---|
| id | bigint | 主键 |
| category_id | bigint | 分类 ID |
| name | varchar(200) | 商品名称 |
| cover_image | varchar(255) | 封面图 |
| short_desc | varchar(500) | 简短说明 |
| content | text | 商品详情 |
| price | decimal(10,2) | 价格 |
| stock | int | 库存 |
| contact_id | bigint | 默认联系方式 ID，可空 |
| is_recommended | tinyint | 是否推荐 |
| sort_order | int | 排序值 |
| status | tinyint | 状态，1上架0下架 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

### 3.4 product_images
| 字段名 | 类型 | 说明 |
|---|---|---|
| id | bigint | 主键 |
| product_id | bigint | 商品 ID |
| image_url | varchar(255) | 图片地址 |
| sort_order | int | 排序值 |
| created_at | datetime | 创建时间 |

### 3.5 contacts
| 字段名 | 类型 | 说明 |
|---|---|---|
| id | bigint | 主键 |
| type | varchar(30) | 类型，如 wechat、qq、telegram、email、link |
| name | varchar(100) | 显示名称 |
| value | varchar(255) | 联系值，如微信号、QQ号、链接地址 |
| qr_image | varchar(255) | 二维码图片，可空 |
| jump_url | varchar(255) | 跳转链接，可空 |
| display_places | varchar(255) | 展示位置，如 home,detail |
| sort_order | int | 排序值 |
| status | tinyint | 状态，1启用0禁用 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

### 3.6 site_settings
| 字段名 | 类型 | 说明 |
|---|---|---|
| id | bigint | 主键 |
| setting_key | varchar(100) | 配置键 |
| setting_value | text | 配置值 |
| group_name | varchar(50) | 分组，如 seo、site、home |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

### 3.7 home_sections
| 字段名 | 类型 | 说明 |
|---|---|---|
| id | bigint | 主键 |
| section_key | varchar(50) | 模块标识，如 banner、notice、intro |
| title | varchar(200) | 模块标题 |
| content | text | 模块内容 |
| image_url | varchar(255) | 模块图片 |
| extra_json | text | 扩展配置 JSON |
| sort_order | int | 排序值 |
| status | tinyint | 状态 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

## 4. 关系说明
- `product_categories` 1 对多 `products`
- `products` 1 对多 `product_images`
- `contacts` 可被 `products.contact_id` 关联为默认联系入口
- `site_settings` 用于统一存储站点标题、关键词、描述等配置
- `home_sections` 用于管理首页 Banner、公告、简介等内容块

## 5. 索引建议
- `products.category_id`
- `products.status`
- `products.is_recommended`
- `products.sort_order`
- `product_categories.sort_order`
- `contacts.status`
- `site_settings.setting_key`
- `home_sections.section_key`

## 6. 二期预留表
如后续扩展支付和自动发货，可新增：
- `orders`
- `order_items`
- `payments`
- `delivery_records`
- `users`

## 7. 备注
- 当前设计优先保证简单够用
- 如后续商品结构复杂，可将商品属性拆表
- 如后续支持多管理员角色，可增加角色权限表
