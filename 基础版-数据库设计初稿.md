# 基础版数据库设计初稿

## 1. 设计说明

本期为基础展示版，数据库以内容管理和商品展示为主，不包含支付、订单、自动发货。
数据库使用 PostgreSQL，迁移脚本位于：
- `database/migrations/` - 源迁移脚本
- `backend/src/main/resources/db/migration/` - Flyway 执行目录

## 2. 数据表清单

| 表名 | 说明 | 迁移脚本 |
|---|---|---|
| admin_users | 管理员用户表 | V001 |
| product_categories | 商品分类表 | V002 |
| contacts | 联系方式表 | V003 |
| products | 商品表 | V004 |
| product_images | 商品图片表 | V005 |
| site_settings | 站点配置表 | V006 |
| home_sections | 首页区块表 | V007 |

## 3. 表结构详述

### 3.1 admin_users (管理员用户表)

| 字段名 | 类型 | 约束 | 说明 |
|---|---|---|---|
| id | BIGSERIAL | PK | 主键ID |
| username | VARCHAR(50) | NOT NULL, UNIQUE | 登录用户名 |
| password_hash | VARCHAR(255) | NOT NULL | 密码哈希值(BCrypt) |
| nickname | VARCHAR(50) | - | 显示昵称 |
| status | SMALLINT | NOT NULL, DEFAULT 1, CHECK IN (0,1) | 状态: 0=禁用, 1=启用 |
| last_login_at | TIMESTAMP | - | 最后登录时间 |
| created_at | TIMESTAMP | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 更新时间 |

**索引：**
- `idx_admin_users_status` ON status

**触发器：**
- 自动更新 `updated_at` 字段

### 3.2 product_categories (商品分类表)

| 字段名 | 类型 | 约束 | 说明 |
|---|---|---|---|
| id | BIGSERIAL | PK | 主键ID |
| name | VARCHAR(100) | NOT NULL | 分类名称 |
| slug | VARCHAR(100) | NOT NULL, UNIQUE | 分类别名(URL友好) |
| sort_order | INTEGER | NOT NULL, DEFAULT 0, CHECK >= 0 | 排序序号 |
| status | SMALLINT | NOT NULL, DEFAULT 1, CHECK IN (0,1) | 状态: 0=禁用, 1=启用 |
| created_at | TIMESTAMP | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 更新时间 |

**索引：**
- `idx_product_categories_status` ON status
- `idx_product_categories_sort_order` ON sort_order

### 3.3 contacts (联系方式表)

| 字段名 | 类型 | 约束 | 说明 |
|---|---|---|---|
| id | BIGSERIAL | PK | 主键ID |
| type | VARCHAR(30) | NOT NULL, CHECK IN (...) | 类型: wechat/qq/phone/email/website/other |
| name | VARCHAR(100) | NOT NULL | 显示名称 |
| value | VARCHAR(255) | NOT NULL | 联系值 |
| qr_image | VARCHAR(255) | - | 二维码图片URL |
| jump_url | VARCHAR(255) | - | 跳转链接 |
| display_places | VARCHAR(255) | - | 展示位置(home,product,detail) |
| sort_order | INTEGER | NOT NULL, DEFAULT 0, CHECK >= 0 | 排序序号 |
| status | SMALLINT | NOT NULL, DEFAULT 1, CHECK IN (0,1) | 状态 |
| created_at | TIMESTAMP | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 更新时间 |

**索引：**
- `idx_contacts_type` ON type
- `idx_contacts_status` ON status
- `idx_contacts_sort_order` ON sort_order

### 3.4 products (商品表)

| 字段名 | 类型 | 约束 | 说明 |
|---|---|---|---|
| id | BIGSERIAL | PK | 主键ID |
| category_id | BIGINT | NOT NULL, FK | 所属分类ID |
| name | VARCHAR(200) | NOT NULL | 商品名称 |
| cover_image | VARCHAR(255) | - | 封面图片URL |
| short_desc | VARCHAR(500) | - | 简短描述 |
| content | TEXT | - | 详情内容(富文本) |
| price | NUMERIC(10,2) | NOT NULL, DEFAULT 0, CHECK >= 0 | 当前价格 |
| original_price | NUMERIC(10,2) | CHECK >= 0 OR NULL | 原价(可选) |
| stock | INTEGER | NOT NULL, DEFAULT 0, CHECK >= 0 | 库存数量 |
| contact_id | BIGINT | FK, ON DELETE SET NULL | 联系方式ID(可选) |
| is_recommended | SMALLINT | NOT NULL, DEFAULT 0, CHECK IN (0,1) | 是否推荐 |
| sort_order | INTEGER | NOT NULL, DEFAULT 0, CHECK >= 0 | 排序序号 |
| status | SMALLINT | NOT NULL, DEFAULT 1, CHECK IN (0,1) | 状态: 0=下架, 1=上架 |
| created_at | TIMESTAMP | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 更新时间 |

**外键：**
- `fk_products_category` → product_categories(id) ON DELETE RESTRICT
- `fk_products_contact` → contacts(id) ON DELETE SET NULL

**索引：**
- `idx_products_category_id` ON category_id
- `idx_products_status` ON status
- `idx_products_is_recommended` ON is_recommended
- `idx_products_sort_order` ON sort_order
- `idx_products_name` GIN (全文搜索)

## 4. 关系说明

```
product_categories (1) ←→ (N) products
products (1) ←→ (N) product_images
contacts (1) ←→ (N) products (可选关联)
```

- `products.category_id` 必须关联有效分类
- `products.contact_id` 可选关联联系方式
- 删除分类时阻止删除关联商品 (RESTRICT)
- 删除商品时自动删除关联图片 (CASCADE)
- 删除联系方式时商品引用置空 (SET NULL)

## 5. 触发器说明

所有包含 `updated_at` 字段的表都配置了自动更新触发器：
```sql
CREATE TRIGGER trg_{table}_updated_at
    BEFORE UPDATE ON {table}
    FOR EACH ROW
    EXECUTE FUNCTION update_updated_at_column();
```
